# 🔥 Advanced Recon Subdomain Scanner

## 📌 Project Overview

This project is a Python-based reconnaissance tool designed for bug bounty hunting and ethical hacking practice.

It automates subdomain discovery and live host detection to help map the attack surface of a target domain.

---

## 🎯 Purpose

- Discover subdomains of a target domain  
- Check which subdomains are alive  
- Analyze HTTP status codes  
- Support bug bounty reconnaissance phase  
- Practice Python automation in cybersecurity  

---

## 🛠 Features

- Subdomain brute force scanning  
- Live host detection  
- HTTP status code checking  
- Fast recon automation  
- Lightweight Python script
